About AM Robo

A-M Robo is a prototype developed as a miniature project for further upgrades. The hardware is designed like a war tank. With our own designed shock absorbing tank track system the robo has an ability to travel through tough terrains, like war fronts, ice, mud etc… The robo’s brain or the control system now consist of two microcontrollers: an arduino and a nodemcu. For heavy uses we can use embedded systems. The robo right now has an extra payload capacity of 7kg and battery backup of 20-30 min in a single charge. Connectivity uses Wifi network systems (range upto 500meters) now, we can shift to any type of connectivity according to requirements.for autonomous function it uses GPS system. It includes sensors like ultrasonic sensor, pir sensor, IR sensor for multiple functions.
